# Serverless URL Shortener

Serverless URL shortener built with DynamoDB and Python. No EC2, no idle cost.

## What It Does
- Generates unique 6 character short codes
- Stores mappings in DynamoDB
- Tracks click counts per URL
- Runs completely serverless

## Results
- 5 URLs shortened
- Click tracking working
- Cost: ~$0.00 on free tier

## Tech Stack
- Python 3
- AWS DynamoDB
- boto3
